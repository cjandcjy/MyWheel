# MyWheel
simple demos

## Tabs目录
Tabs组件代码

## Image-Picker目录
Image上传组件代码，server.js是后台代码，用于接收和下载上传的图片

## Scroll目录
原生js实现滚动事件，并且实验了防抖、 节流等方法降低浏览器渲染的卡顿现象

##CssWave目录
纯css实现波浪效果，scss文件利用node-sass工具转换为css文件
进入node_modules/bin目录 ， 运行 node-sass --watch ../../CssWave/index.scss ../../CssWave/index.css 即可
